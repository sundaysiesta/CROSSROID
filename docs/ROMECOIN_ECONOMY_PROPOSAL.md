# ロメコイン経済システム提案書

## 📊 ロメコインの価値分析（実データベース）

### 実際のメッセージ数データ
- **上位10人: 週間3,000-4,000メッセージ**
- **15位まで: 週間2,000メッセージ**
- **30位まで: 週間1,000メッセージ以上**

### ロメコイン獲得量の計算

**1メッセージあたりの獲得量:**
- 基本: 10コイン
- 会話参加者数ボーナス: 1 + (参加者数/10) → 最大2倍
- 深夜ボーナス（6時前）: 1.5倍
- 返信: +5コイン
- リアクション: +5コイン

**平均獲得量: 15-20コイン/メッセージ**（会話参加者数や深夜の影響を考慮）

### 実際の獲得量（週間）

**上位10人（週間3,000-4,000メッセージ）:**
- **週間獲得: 45,000-80,000コイン**
- **1日: 約6,400-11,400コイン**
- **月間: 約193,000-343,000コイン**

**15位まで（週間2,000メッセージ）:**
- **週間獲得: 30,000-40,000コイン**
- **1日: 約4,300-5,700コイン**
- **月間: 約129,000-171,000コイン**

**30位まで（週間1,000メッセージ以上）:**
- **週間獲得: 15,000-20,000コイン以上**
- **1日: 約2,100-2,900コイン以上**
- **月間: 約64,000-86,000コイン以上**

**カジュアルユーザー（週間100メッセージ程度）:**
- **週間獲得: 1,500-2,000コイン**
- **月間: 約6,400-8,600コイン**

### 現在の使用例
- じゃんけん勝負のベット: **最低100コイン**（上位10人なら1日の獲得量の約1-2%）

## 📊 月間ランキング賞金システム（実装案）

### 概要
一カ月の間にどれくらい話したかのランキング（レベルランキング）で、上位10人にロメコインを配布します。

**注意:** 既にたくさん話している人が上位になるため、賞金額は控えめに設定し、富の集中を防ぎます。

### 賞金額（月間、上位10人に配布）

| 順位 | 賞金額 | 根拠 |
|:---|:---|:---|
| **1位** | **15,000コイン** | 30位までの週間獲得量に相当。控えめな設定で富の集中を防止 |
| **2位** | **12,000コイン** | 30位までの週間獲得量の0.6-0.8倍 |
| **3位** | **10,000コイン** | 部活ランキング1位の賞金と同額 |
| **4位** | **8,000コイン** | 部活ランキング3位の賞金と同額 |
| **5位** | **6,000コイン** | 部活ランキング5位の賞金と同額 |
| **6位** | **5,000コイン** | 30位までの週間獲得量の0.25-0.33倍 |
| **7位** | **4,000コイン** | 30位までの週間獲得量の0.2-0.27倍 |
| **8位** | **3,000コイン** | 部活維持費と同額 |
| **9位** | **2,500コイン** | 控えめな報酬 |
| **10位** | **2,000コイン** | 控えめな報酬 |

**合計賞金額: 67,500コイン/月**

### 賞金システムの効果

1. **控えめな報酬で富の集中を防止**
   - 既にたくさん話している人が上位になるため、賞金額は控えめ
   - 1位でも30位までの週間獲得量程度で、過度な富の集中を防ぐ

2. **継続的な活動のインセンティブ**
   - 月間で話した量に応じた報酬
   - 上位10人に入ることで、少しの追加ロメコインを獲得

3. **経済循環の促進**
   - 月1回の賞金配布によるロメコインの循環
   - 控えめな賞金額で、経済バランスを維持

## 🏆 人気者・嫌われ者選手権賞金システム（実装案）

### 概要
人気者・嫌われ者選手権は、アクティブ度とは相関があってもそこまで強くないイベントです。
そのため、賞金額を高めに設定し、参加意欲を高めます。

### 賞金額（イベント終了時、上位入賞者に配布）

| 順位 | 賞金額 | 根拠 |
|:---|:---|:---|
| **1位** | **50,000コイン** | 上位10人の週間獲得量の0.6-1.1倍。高めの設定で参加意欲を高める |
| **2位** | **40,000コイン** | 上位10人の週間獲得量の0.5-0.9倍 |
| **3位** | **30,000コイン** | 15位までの週間獲得量に相当 |
| **4位** | **25,000コイン** | 30位までの週間獲得量の1.25-1.67倍 |
| **5位** | **20,000コイン** | 30位までの週間獲得量の1-1.33倍 |
| **6-10位** | **15,000コイン** | 30位までの週間獲得量に相当 |
| **参加賞** | **3,000-5,000コイン** | 参加者全員に配布 |

**合計賞金額: 約200,000-300,000コイン/イベント**（参加者数による）

### 賞金システムの効果

1. **高めの賞金額で参加意欲を高める**
   - アクティブ度とは相関があってもそこまで強くないため、高めの賞金額を設定
   - 1位なら上位10人の週間獲得量の0.6-1.1倍で、十分なインセンティブ

2. **公平な競争の促進**
   - アクティブ度だけでなく、人気度やコミュニティへの貢献度も評価
   - 様々なユーザーが上位に入る可能性がある

3. **経済循環の促進**
   - 高めの賞金額で、ロメコインの循環を促進
   - イベントの特別感を演出

## 🏛️ 部活チャンネル維持費とランキング賞金システム（実装案）

### 基本設定

#### 部活作成費
- **初回作成費: 10,000コイン**（現在の設定を維持）
- **根拠:**
  - 30位までのユーザーの週間獲得量（15,000-20,000コイン）の0.5-0.67倍
  - アクティブでないユーザーでも作成可能な適度な障壁
  - 気軽に部活を作れるが、真剣さは必要

#### 週間維持費（部長が1人で支払い）
- **週間維持費: 3,000コイン**
- **根拠:**
  - 30位までのユーザーの週間獲得量（15,000-20,000コイン）の0.15-0.2倍
  - 部長が1人で支払っても週間獲得量の15-20%程度で、負担が軽い
  - 使われない部活を減らすための適度な障壁
  - 話していれば稼げる金額で、アクティブな部活なら問題なく支払える

### 部活ランキングTOP5賞金システム

**目的:** 人気部活（TOP5）の部長に賞金を支払い、部活運営のインセンティブを提供

#### 賞金額（週間、部長に支払い）

| 順位 | 賞金額 | 実質収益 | 根拠 |
|:---|:---|:---|:---|
| **1位** | **10,000コイン** | **+7,000コイン** | 維持費（3,000）の約3.3倍。元が取れる |
| **2位** | **9,000コイン** | **+6,000コイン** | 維持費の3倍。維持費+6,000コイン |
| **3位** | **8,000コイン** | **+5,000コイン** | 維持費の約2.67倍。維持費+5,000コイン |
| **4位** | **7,000コイン** | **+4,000コイン** | 維持費の約2.33倍。維持費+4,000コイン |
| **5位** | **6,000コイン** | **+3,000コイン** | 維持費の2倍。維持費+3,000コイン |

**合計賞金額: 40,000コイン/週**

**注意:** 毎週もらえるものなので、話していれば稼げる金額を考慮した設定

#### 賞金システムの効果

1. **TOP5入りすれば維持費以上の収益**
   - 1位: 維持費3,000 → 賞金10,000 = **+7,000コイン**（元が取れる）
   - 2位: 維持費3,000 → 賞金9,000 = **+6,000コイン**
   - 3位: 維持費3,000 → 賞金8,000 = **+5,000コイン**
   - 4位: 維持費3,000 → 賞金7,000 = **+4,000コイン**
   - 5位: 維持費3,000 → 賞金6,000 = **+3,000コイン**

2. **部活運営のモチベーション向上**
   - 人気部活になることで経済的メリット
   - TOP5に入れば全て維持費以上の収益
   - 1位なら維持費の約3.3倍で、元が取れる
   - 話していれば稼げるので、アクティブな部活なら維持可能

3. **使われない部活の自然淘汰**
   - TOP5に入れない部活は維持費のみの負担（週3,000コイン）
   - 非アクティブな部活は自然に廃部

### 自動徴収と廃部システム

#### 週1回の自動処理（部活ランキングソート時）

1. **部活ランキングの計算**
   - 部活チャンネルのメッセージ数やアクティビティでランキングを算出
   - TOP5を決定

2. **維持費の自動徴収**
   - 各部活の部長から週間維持費（3,000コイン）を徴収
   - 部長のロメコイン残高を確認

3. **TOP5賞金の支払い**
   - ランキング1-5位の部長に賞金を支払い
   - 賞金額は維持費徴収後に加算

4. **廃部判定**
   - **即廃部条件:**
     - 部長が存在しない（部長ロールなし、退会など）
     - 部長のロメコイン残高が維持費（3,000コイン）未満
   - 廃部時: チャンネルをアーカイブまたは削除

#### 処理フロー例

```
週間ランキング更新時:
1. 各部活のアクティビティを集計
2. ランキングTOP5を決定
3. 各部活の部長を確認
   ├─ 部長不在 → 即廃部
   └─ 部長存在 → 残高確認
       ├─ 3,000コイン未満 → 即廃部
       └─ 3,000コイン以上 → 維持費徴収
           ├─ TOP5入り → 賞金支払い
           └─ TOP5外 → 維持費のみ徴収
```

### 経済バランスの考慮

#### 週間のコイン循環

**徴収:**
- 各部活から3,000コイン/週
- 仮に20部活ある場合: 60,000コイン/週

**支払い:**
- TOP5賞金: 40,000コイン/週
- 純徴収: 約20,000コイン/週（20部活の場合）

**効果:**
- 使われない部活が減ることで、徴収総額は減少
- TOP5の部活は実質的に維持費が無料またはプラス
- 経済循環が促進される

#### ユーザー体験への影響

**メリット:**
- 人気部活の部長に報酬が支払われる
- 使われない部活が自然に淘汰される
- 部活運営のモチベーション向上

**注意点:**
- 部長の負担は週3,000コイン（話していれば稼げる金額）
- TOP5に入れない部活は維持費のみの負担（週3,000コイン）
- 部長が不在または資金不足（3,000コイン未満）の場合、即廃部となる

- **30位まで**: 週間15,000-20,000コイン獲得 → 小規模イベント1位は週間獲得量の1.25-1.67倍

これらの設定により、実際のユーザー活動量に基づいた適切な経済バランスを実現し、ロメコイン経済の健全な循環と、ユーザーのモチベーション維持を両立できます。
