# ロメコインシステム 手数料・税について

## 📋 概要

ロメコインシステムでは、現在手数料・税は実装されていません。

以前はインフレ対策として以下の手数料・税が実装されていましたが、現在はすべて削除されています：

- 取引手数料（1%）- `/give`、`/duel`、`/duel_russian`コマンド
- 消費税（5%）- ショップでの購入
- パリミュチュエル手数料（25%）- `/race bet`コマンド

## 🔮 将来の実装用（未実装）

### インフレ税（段階的税率）

**税率**: 所持金に応じた段階的税率
- 100万以上: 0.1%/日
- 500万以上: 0.2%/日
- 1000万以上: 0.5%/日

**適用箇所**: 未実装（将来の実装用）

**計算方法**:
- 所持金 × 税率 = 1日あたりの税額
- 課税されたロメコインはシステムから削除される

**実装ファイル**: `features/inflationControl.js`（関数は実装済み、定期実行は未実装）
